## 🚀 Featured Projects

### expression-parser

Parses mathematical expressions from raw strings using a handwritten lexer, recursive descent parsing, and abstract syntax tree evaluation.

**Technical Focus:** Language processing, tokenization, AST construction, recursive algorithms

### spell-checker

High-performance spell-checker built around a custom hash table implementation using separate chaining instead of standard library collections.

**Technical Focus:** Hash functions, data structures, memory efficiency, lookup optimization

### maze-explorer

2D tile-based maze game featuring sprite animation, a dynamic scrolling camera, and a proximity-based enemy line-of-sight tracking system.

**Technical Focus:** Game loops, collision detection, state management, rendering pipelines

### rps-machine

Real-time Rock Paper Scissors game using OpenCV and TensorFlow for webcam-based gesture classification and machine learning inference.

**Technical Focus:** Computer vision, ML model inference, webcam processing

### connect4

Interactive Connect 4 application using custom Java Swing rendering, alpha composited hover previews, and win highlighting.

**Technical Focus:** GUI programming, custom rendering, event handling, matrix traversal

### connect4-python

Event-driven Connect 4 built with Python and cmu_graphics, featuring 2D board state management and win-detection algorithms.

**Technical Focus:** Event-driven programming, UI state management
